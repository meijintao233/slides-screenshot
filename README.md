# slides-screenshot

export local slides

---

## Install

```
npm i mulit-article-publisher
```

---

## Usage

```javascript
screenshot({
  slidesUrl: "https://slides.com/charliemei/deck",
  publicPath: `./slides`
});
```

---

## API

- screenshot(options)

  options:

`slidesUrl` :string --- the slides path

`publicPath` :string --- local diretory
